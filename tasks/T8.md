# T8 — Three-Day Preview Strip

## Shared Context

You are working on **sunset-color**, a one-page website that predicts the color of the sky at sunset for a US location and renders the prediction as the page background.

**Concept:** the forecast IS the design. A 3-stop vertical gradient shows predicted sunset colors. Drifting wispy SVG shapes show forecast cloud cover at high/mid/low altitudes.

**Stack:** Next.js 14+ (App Router), TypeScript, Tailwind CSS.

**Where this fits:** the main page (T7) shows today's (or tomorrow's) sunset prediction. This task adds a small strip at the bottom showing the next three days, so users can plan around a good one.

---

## Task: Three-Day Preview Strip

**Goal:** A small horizontal row of swatches at the bottom of the main page showing the next three days. Clicking a swatch swaps it into the main view.

**Depends on:** T1, T2, T7.

**Files to create:**
- `components/MultiDayStrip.tsx`
- Wire it into `app/page.tsx` (small modification).

**Interface contract:**
```ts
export function MultiDayStrip(props: {
  lat: number;
  lng: number;
  onSelectDate: (date: string) => void;   // YYYY-MM-DD
  inkColor: string;
}): JSX.Element;
```

**Behavior:**
- On mount (or when lat/lng changes), fetch `/api/forecast` three times for the three upcoming days (skip "today" — that's already shown in the main view).
- Render three circles, each ~32px, filled with the predicted dominant color (the middle gradient stop from `forecastToColor`).
- Beneath each circle: tiny day label ("Tue", "Wed", "Thu") in Inter caps, opacity 0.6, using `inkColor`.
- Hover → swatch scales 1.1x with 200ms ease.
- Click → calls `onSelectDate`, parent (T7) refetches and animates to that day's gradient. The hero time updates to that date and the strip label adjusts ("Mon" for the new "tonight", etc.).

**Layout:**
- Anchored bottom-center, ~48px from viewport bottom.
- Horizontal flex with ~24px gap between swatches.

**Acceptance criteria:**
- Strip appears once the main forecast loads.
- Three swatches with distinct colors visible (assuming varying weather).
- Clicking the second swatch transitions main view to that day's gradient; hero time/date updates.
- Swatches re-fetch when the user selects a different location.
