# T2 — Color & Gradient Mapping

## Shared Context

You are working on **sunset-color**, a one-page website that predicts the color of the sky at sunset for a US location and renders the prediction as the page background.

**Concept:** the forecast IS the design. A 3-stop vertical gradient shows predicted sunset colors. Drifting wispy SVG shapes show forecast cloud cover at high/mid/low altitudes. A small italic descriptor ("fiery peach", "muted", "overcast", etc.) sits below the location name and sunset time.

**Stack:** Next.js 14+ (App Router), TypeScript, Tailwind CSS, deployed on Vercel. US-only scope.

**Physics summary (this task implements this):** Four ingredients drive sunset color:
1. **High clouds present** (the canvas) — sweet spot 30–70% high-cloud cover.
2. **Clear lower atmosphere** — clean air = vivid colors. Haze/pollution mutes.
3. **Moderate humidity** — high humidity = haze near the ground.
4. **Narrow aerosol range** — a *small* amount of PM2.5 deepens reds; too much mutes everything.

## Task: Color & Gradient Mapping

**Goal:** A pure, well-tested function that turns a `ForecastResult` into a 3-stop vertical gradient + descriptor + score. No fetch, no DOM, no React — just data in, data out.

**Depends on:** T1 (for the `ForecastResult` type only — import it from `../lib/forecast`).

**Files to create:**
- `lib/colorMap.ts`
- `lib/colorMap.test.ts`

**Interface contract:**
```ts
import type { ForecastResult } from './forecast';

export type GradientStop = { color: string; position: number }; // position 0=top, 1=bottom
export type ColorResult = {
  gradient: { stops: [GradientStop, GradientStop, GradientStop] }; // top, middle, bottom
  descriptor: string;       // e.g., "fiery peach", "blood red", "muted"
  score: number;            // 0..1, overall quality
};

export function forecastToColor(f: ForecastResult): ColorResult;
```

**The formula (implement this, then tune by eye):**

1. **Score (0..1)** — weighted product:
   - `high_cloud_factor`: tent function peaking at 50% high cloud, 0 at 0% or 100%. Roughly `1 - |0.5 - high_cloud_pct/100| * 2`.
   - `low_cloud_penalty`: `1 - (low_cloud_pct/100) * 0.7`. Heavy low cloud blocks the show.
   - `aerosol_factor`: bell curve over PM2.5. Peak around 8 µg/m³, drops sharply above 35 µg/m³. If PM2.5 is null, fall back to visibility: >15km → 0.9, <5km → 0.3.
   - `humidity_penalty`: `1 - max(0, (humidity_pct - 60)/100)` — only penalize above 60% RH.
   - `score = high_cloud_factor * low_cloud_penalty * aerosol_factor * humidity_penalty`

2. **Dominant hue family** — pick based on (humidity, PM2.5, score):
   - Low humidity + low PM2.5 + high score → orange family (`#ff8c42`, `#ffb380`)
   - Medium humidity + low PM2.5 + high score → pink/rose family (`#ff6b8a`, `#ffb3c1`)
   - Moderate PM2.5 (15–35) → deep red / magenta (`#c0392b`, `#8e44ad`)
   - High PM2.5 (>50) → blood red / purple, muted (`#7b1f1a`, `#553344`)
   - Low score (any cause) → grey family (`#8a8a8a`, `#5a5a5a`)
   - Overcast (`low_cloud_pct > 80`) → slate grey regardless (`#6c7680`, `#3d4248`)

3. **Gradient stops** — given a dominant hue:
   - Top stop (position 0): pale washed version (high lightness, low saturation).
   - Middle stop (position 0.55): dominant color at full saturation.
   - Bottom stop (position 1): deep horizon color (low lightness, slightly red-shifted).
   - Work in HSL space, output `#rrggbb` hex.

4. **Descriptor** — pick from a small bank based on (hue family, score, low_cloud_pct):
   - Words: "fiery", "peach", "rose", "magenta", "blood red", "smoky", "muted", "pale", "overcast", "still", "amber", "lilac".
   - High score + orange → "fiery peach" or "amber"
   - High score + pink → "rose" or "pink"
   - Moderate score → "muted"
   - Overcast → "overcast" or "still"
   - High PM2.5 → prepend "smoky" (e.g., "smoky red")

**Test cases for `colorMap.test.ts`:**
- Ideal sunset (40% high cloud, 10% low cloud, PM2.5=8, humidity 45%, vis 30km) → orange family, score > 0.8, descriptor includes "fiery" or "peach".
- Overcast (90% low cloud, anything else) → grey gradient, descriptor "overcast".
- Wildfire smoke (PM2.5 = 80) → muted red/purple, descriptor includes "smoky".
- Clear sky no clouds (high=0, mid=0, low=0) → low score, pale descriptor.
- Boundary: high_cloud_pct = 50 → near-peak high_cloud_factor.
