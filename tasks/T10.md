# T10 — Share Button + Production Deploy

## Shared Context

You are working on **sunset-color**, a one-page website that predicts the color of the sky at sunset for a US location and renders the prediction as the page background.

**Concept:** the forecast IS the design. A 3-stop vertical gradient shows predicted sunset colors. Drifting wispy SVG shapes show forecast cloud cover at high/mid/low altitudes.

**Stack:** Next.js 14+ (App Router), TypeScript, Tailwind CSS, deployed on Vercel.

**Share strategy:** uses the `/api/share` endpoint (built in T6) which returns a 1080×1080 PNG. The button on the main page lets users download/share it.

---

## Task: Share Button + Production Deploy

**Goal:** Add a share button to the main page that triggers download/copy of the share image, and ensure the production deploy is healthy.

**Depends on:** T6, T7.

**Tasks:**

1. **Add share button to `app/page.tsx`.**
   - Small circular icon button in the bottom-right corner (~40px), semi-transparent backdrop in current `inkColor`.
   - Icon: simple share/arrow SVG.
   - Only visible after a location has been selected.

2. **On click:**
   - `fetch('/api/share?lat=...&lng=...&name=...&date=...')` → blob.
   - If `navigator.share()` and `navigator.canShare({ files: [...] })` are supported (mobile): use that.
   - Else: trigger download via a temporary `<a download>` element.

3. **Production deploy verification:**
   - Confirm Vercel deploy succeeds and the live URL works end-to-end: search a city, see gradient + clouds, share button downloads a working PNG.
   - Verify `AIRNOW_API_KEY` is set in Vercel for Production.
   - Open browser DevTools network tab on the live site, confirm the AirNow key is never visible in any response.

4. **Add `<meta>` tags to `app/layout.tsx`.**
   - `title`: "sunset-color"
   - `description`: "predicted sunset colors, by location"
   - `og:image`: `/api/share?lat=37.77&lng=-122.42&name=San+Francisco` (default to San Francisco)
   - `og:title`, `og:description` to match.
   - `twitter:card`: "summary_large_image"

**Acceptance criteria:**
- Share button on production downloads a PNG. PNG matches on-screen gradient (clouds aside).
- Sharing the prod URL on iMessage/Twitter shows a sunset preview thumbnail (og:image works).
- No console errors on production.
- AirNow key never exposed in any HTTP response on the live site.

---

Execute only this task. Stop when all acceptance criteria are met. Commit and push when done.
