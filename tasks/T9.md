# T9 — Grey/Overcast Edge Cases + Descriptor Polish

## Shared Context

You are working on **sunset-color**, a one-page website that predicts the color of the sky at sunset for a US location and renders the prediction as the page background.

**Concept:** the forecast IS the design. A 3-stop vertical gradient shows predicted sunset colors. Drifting wispy SVG shapes show forecast cloud cover at high/mid/low altitudes.

**Design principles relevant here:**
- Honest rendering: grey/overcast days shown as grey with a melancholy descriptor — not faked into something pretty.
- Typography stays poetic even on grey days; the experience is still beautiful.

**Stack:** Next.js 14+ (App Router), TypeScript, Tailwind CSS.

---

## Task: Grey/Overcast Edge Cases + Descriptor Polish

**Goal:** Make sure the website handles low-quality forecasts gracefully and that descriptors feel coherent.

**Depends on:** T2, T7.

**Tasks:**

1. **Audit `lib/colorMap.ts`'s descriptor bank.** Ensure every score range and hue family has a word that fits. Add cases to `lib/colorMap.test.ts` for:
   - Pure overcast (low_cloud_pct=95, others neutral).
   - Foggy (visibility=1km).
   - Wildfire smoke (PM2.5=120, visibility=8km).
   - Light haze (PM2.5=25, humidity=75%).

2. **Add a poetic second line for low scores.** In `app/page.tsx`, when `score < 0.3`, render a second line below the descriptor in italic, opacity 0.6, picking from a small bank:
   - "stay in tonight"
   - "tomorrow then"
   - "the sky is resting"
   - "muted but yours"
   - "soft grey"
   - "no fireworks tonight"

   Pick deterministically based on `score` (hash to index) so the same forecast always shows the same line — don't re-randomize on re-renders.

3. **Verify `pm25_ugm3: null` path.** Read `lib/colorMap.ts` and confirm the visibility-based aerosol fallback works. Add a test case for it.

**Acceptance criteria:**
- All new test scenarios pass.
- Manually verify in browser via mocked forecast data (you can stub `getSunsetForecast` temporarily or visit `/api/forecast` for a real city known to be overcast):
  - Ideal sunset → coherent fiery typography + color.
  - Overcast → grey + "overcast" + poetic second line.
  - Smoky → muted red + "smoky red" descriptor.
  - Foggy → grey-toned + poetic second line.
- Nothing renders as `NaN`, `undefined`, or empty string.
